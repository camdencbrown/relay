# Credentials Needed for Demo

## Salesforce (Required for Demo)

Get these from your Salesforce account:

1. **Username:** your@email.com
2. **Password:** yourpassword
3. **Security Token:** 
   - Go to: Setup → Personal Setup → My Personal Information → Reset My Security Token
   - Token will be emailed to you
   - Format: `AbCdEfGh1234567890`

**Test connection:**
```python
from simple_salesforce import Salesforce
sf = Salesforce(
    username='your@email.com',
    password='yourpassword',
    security_token='yourtoken'
)
print("Connected!" if sf else "Failed")
```

## AWS (Configure via .env file)

Set up your AWS credentials in `.env` file:

```bash
AWS_ACCESS_KEY_ID=your_access_key_here
AWS_SECRET_ACCESS_KEY=your_secret_key_here
AWS_DEFAULT_REGION=us-west-1
S3_BUCKET_NAME=your-bucket-name
```

Copy `.env.example` to `.env` and fill in your credentials.
**Never commit `.env` to git!**
